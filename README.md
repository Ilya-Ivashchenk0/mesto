# Проект: Место

**Описание проекта**

На странице прокта вы можее найти полезную информацию для путешествий по России а так же добавить информацию о себе и своих местах которые вы посещали.

**Технологии**

HTML5 + CSS3 + JavaScript

Grid Layout and Flex Box
